PaiAdmin

PaiAdmin是一款基于TP6框架开发的后台管理系统，系统具有简洁性，便于二次开发。

起步 Gitee Github

主要特性

PaiAdmin是一款基于ThinkPHP6 + Light Year Admin的后台开发框架；系统开发的目的是：不必再为了做底层架构而浪费时间，专注于做业务逻辑，实现项目的快速；简单一句话，花最少的时间赚更多的钱。

完善的权限验证

内置基于Auth验证的权限管理系统，支持父子级继承、支持管理员账号分组、支持子级数据修改。

表单生成器

采用PHP FormBuilder表单生成器,可以快速生成现代化的form表单，内置17种常用的表单组件，支持表单验证。

附件管理系统

内置图片附件管理，支持上传到本地以及云储存COS，还具有可扩展性,可以自行扩展七牛云，阿里云储存等.

公众号&小程序开发

采用EasyWeChat作为组件开发，支持图文管理，微信菜单管理，自动回复功能，支持微信公众号和小程序快速开发。

支付系统模块

采用yansongda/pay最优雅的Alipay 和 WeChat 的支付 SDK 扩展包，支持微信支付宝手机、电脑、小程序、公众号支付。

长连接&定时任务

Workerman是一款开源高性能异步PHP socket即时通讯框架。支持Websocket，内置有PHP定时器，可扩展定时任务。
